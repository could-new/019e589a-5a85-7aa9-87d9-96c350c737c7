# PinoCart App

This is the mobile e-commerce application for [PinoCart.com](https://pinocart.com). 

## Overview
A cross-platform Flutter application tailored for PinoCart's online store. The app currently features a base structural layout for an e-commerce platform, including:
- Responsive Home Screen with promotional banners
- Product Grid displaying featured items
- Shopping Cart interface

## Next Steps for Integration
To connect this app to the live PinoCart store, the following integrations are pending:
1. **API Integration:** Connecting to the website's backend (e.g., WooCommerce/Shopify API) to fetch live products, categories, and inventory.
2. **Authentication:** User login and registration flows.
3. **Checkout & Payments:** Integration with a payment gateway (e.g., Stripe) to process orders securely.
4. **Branding:** Applying exact brand colors, fonts, and logos.

## Tech Stack
- **Framework:** Flutter (Dart)
- **Target Platforms:** Android, iOS, Web

## How to Run
1. Ensure you have the Flutter SDK installed.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your connected device or emulator.
